100好习惯

#项目目标
用“100天法”帮助用户在一年内以非连续方式完成目标，通过倒计时与执行意图（如果-那么）降低行动门槛。

#核心功能
## ✨ 核心功能
- **创建习惯**：最多4个习惯，每个至少3个执行意图
- **打卡**：每天只能打卡一次，必须选择一个意图
- **查看习惯列表**：显示所有习惯和进度
- **统计信息**：查看详细统计和意图使用情况
- **日历视图**：可视化本月打卡记录
